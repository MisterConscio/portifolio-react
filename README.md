# Portifolio em React

Esse projeto faz o uso do React para montar um portifólio baseado em informações do github

## Setup e Desenvolvimento do porjeto

Instalação das ferramentas:

```console
npm install
```

Modo de desenvolivmento:

```console
npm start
```

Modo de teste:

```console
npm test
```

Build do projeto:

```console
npm run build
```
